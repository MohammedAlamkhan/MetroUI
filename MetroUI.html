theming support

Current accent color has been set to Cyan for presentation purposes.

More colors coming with MetroJS implementation, meaning every color of Windows Phone 8 will be supported. Besides that, every UI element supports dark and light backgrounds.

buttons

Buttons are used for temporary interaction, whether in forms or just to run a JavaScript function.
They can easily be inserted using code as shown below.

Example:

Use the button to switch between dark and light backgrounds.

Code - JavaScript:

	<button class="metro" type="button" onclick="alert('You\'ve clicked an active button!')">Button</button>
	<button class="metro" type="button" disabled>Disabled</button>

switches

Switches are used, unlike buttons, for long-term interaction, meaning they can change settings for a longer period than buttons.
They are identified by their "id" attribute, so they can identify their belonging checkbox.
Metro switches containing the class "disabled" are not set up as a working switch.

Example:


Code - HTML:

	<div class="metro checkbox" id="switch1">
	    <div class="innerCheckbox"></div>
	</div>
	
	<div class="metro checkbox" id="switch2">
	    <div class="innerCheckbox"></div>
	</div>

	<div class="metro checkbox disabled" id="switch3">
	    <div class="innerCheckbox"></div>
	</div>


Using the JavaScript code below, you can create a new checkbox and assign it to the MetroUI switch. It binds an event to the switch based on touch/click input toggling its "checked" class and the checkboxes "checked" state.

Code - JavaScript:

	MetroUI.init({
		objID: $('#switch1'), 	// Create new switch indentified by jQuery id selector
		id: "switch1",			// Assign both the switch and a newly created input an ID, should be same as objID
		startEnabled: false	// OPTIONAL: Should the switch be enabled on creation? Default: false
	});

	MetroUI.init({
		objID: $('#switch2'),
		id: "switch2",
		startEnabled: true
	});

	// Will not be set up as MetroUI switch
	MetroUI.init({
		objID: $('#switch3'),
		id: "switch3",
		startEnabled: true
	});

important notice


MetroUI.css's JavaScript is based on jQuery, so be sure to include that before everything else!

Code - HTML:

		<script src="http://code.jquery.com/jquery-1.11.0.min.js"></script> <!-- Use if you want to load jQuery from a remote resource -->
		<script src="javascript/jquery-1.11.0.min.js"></script>				<!-- Use if you have jQuery saved locally, change Source if needed -->
