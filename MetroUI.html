theming support

Current accent color has been set to Crimson for presentation purposes.

More colors coming with MetroJS implementation, meaning every color of Windows Phone 8 will be supported. Besides that, every UI element supports dark and light backgrounds.

buttons

Buttons are used for temporary interaction, whether in forms or just to run a JavaScript function.
They can easily be inserted using code as shown below.

Example:

Use the button to switch between dark and light backgrounds.

Code - JavaScript:

		<button class="metro" type="button" onclick="alert('You\'ve clicked an active button!')">Button</button>
		<button class="metro" type="button" disabled>Disabled</button>

switches

Switches are used, unlike buttons, for long-term interaction, meaning they can change settings for a longer period than buttons.
They are identified by their "data-id", so they can identify their belonging checkbox.

Example:

Checkbox 1 is: checked

Checkbox 2 is: disabled


Code - HTML:

		<input class="metro" type="checkbox" data-id="switch1" checked>
		<div class="metro checkbox checked" data-id="switch1">
			<div class="innerCheckbox"></div>
		</div>
	
		<input class="metro" type="checkbox" data-id="switch2" disabled>
		<div class="metro checkbox disabled" data-id="switch2">
			<div class="innerCheckbox"></div>
		</div>


Using the JavaScript code below, you can use the MetroUI.css switches to toggle their corresponding checkboxes, making it easier to use existing JavaScript with MetroUI.css.

These lines basically bind an event (based on either touch or click) to a switch with an identification attribute of "switch1", toggling it's class "checked" and applying it's changed settings to a Input with the same identification.

Code - JavaScript:

		$('div[data-id=switch1]').bind(touchEventEnd, function() {
			// Storing the requested element in a variable for easier access
			var $checkboxElement = $('input[type=checkbox][data-id=' + $(this).attr("data-id") + ']');
			
			// Applying new class to the MetroUI switch and its corresponding checkbox
			$(this).toggleClass("checked");
			$checkboxElement.attr("checked", !$checkboxElement.attr("checked"));
			
			// PRESENTATION ONLY: Changing the status string for the first MetroUI switch
			if ($checkboxElement.attr("checked") == "checked") {
				$('p#checkbox1checked').html("Checkbox 1 is: " + $checkboxElement.attr("checked"));
			} else {
				$('p#checkbox1checked').html("Checkbox 1 is: unchecked");
			}
		});

important notice


MetroUI.css's JavaScript is based on jQuery, so be sure to include that before everything else!

Code - HTML:

		<script src="http://code.jquery.com/jquery-1.11.0.min.js"></script> <!-- Use if you want to load jQuery from a remote resource -->
		<script src="javascript/jquery-1.11.0.min.js"></script>				<!-- Use if you have jQuery saved locally, change Source if needed -->
