Required Libraries
Libraries

MetroUI relies on third-party libraries. This page is going to show you, which libraries are used and explains their function. It also shows you which libraries you can safely ignore if you are not going to use some features.

In the near future, some libraries will be merged with MetoUI's JavaScript components to reduce the number of HTTP requests.

jQuery



jQuery is the core of everything. It takes care of DOM management in MetroUI for Modals and other things. While including MetroUI to your page, be sure to add jQuery before anything.

<script type="text/javascript" src="js/lib/jquery-1.9.0.min.js"></script>


IScroll

IScroll is responsible for horizontal scrolling in Windows mode and vertical scrolling in Windows Phone mode. If you are going to use your own scrolling solution, you don't need this.

<script type="text/javascript" src="js/lib/iscroll.js"></script>


Fastclick.js

Fastclick.js removes the 300ms delay on click events found in current mobile browsers.

<script type="text/javascript" src="js/lib/fastclick.js"></script>
