App Parameters
 

MetroUI is highly configurable due to many parameters being used while initializing. These parameters allow you to enable or disable certain features, change texts or just the way things are being handled.

Because there are so many parameters that can be passed during initialization, this list shows you which parameter has which effect.

As MetroUI is still being developed, more parameters are being added, making it more user-friendly and controllable.

Available Parameters
Parameter Type Default Description
General
initFastClick boolean true Enables FastClick while initializing MetroUI. Eliminates 300ms Delay on touch devices.
optimizeIpad boolean true Removes 20px bottom padding on iPads with iOS 7.
preventDefault boolean true Prevents scrolling on touch devices. iScroll is used as an alternative instead.
pushHomeToHistory boolean true Adds inital page to navigation history. Useful if using Navigation.
windowSizeLimit int 768 Sets maximum device width for JavaScript components. If changed, don't forget to change this value in MetroUI.less too.
Application Bars
barsRightClick boolean true Normally, MetroUI catches right clicks to toggle Bars. If set to false, you'll get the default context menu instead. Disable this if you're going to use the default context menu on your website.
barsOnOpen string 'both' Specifies which bar should be opened on Right click. Disabled with barsRightClick: false. Values: 'appBar', 'navBar', 'both'
closeBarsOnIconClick boolean false If true, Application Bars will be closed when
a) an AppBar icon is clicked or
b) a link in Navigation bar is clicked.
Forms
select boolean true If false, MetroUI won't initialize select lists and they will not work in any way. Set this to false if you don't use select lists anywhere.
switchDynamics boolean true When using Switches, you can also drag them instead of clicking them. Set this to false if you're experiencing performance issues.
Modals
modalTitle string 'MetroUI' Specifies the default title of Alert and Confirms.
modalOkTitle string 'ok' Specifies the text that is displayed on OK Buttons in Alerts and Confirms.
modalCancelTitle string 'cancel' Specifies the text that is displayed on Cancel Buttons in Alerts and Confirms.
Notifications
notificationDuration int 5000 Specifies the time a notification is shown before it disappears.
Page Transitions
pageTransitions boolean true Enables or disables page transition animations.
pageTransitionInOnStart boolean true Enables or disables page transition animations on start.
pageCreateScroll boolean true Enables or disables creating iScroll for pages after transition.
iScroll - More info here
scrollX boolean true Enables or disables horizontal scrolling.
scrollY boolean false Enables or disables vertical scrolling.
mouseWheel boolean true Enables or disables scrolling with mouse wheel.
disableMouse boolean true Enables or disables mouse dragging.
scrollbars boolean, string 'custom' Sets Scrollbar visibility and custom class. Values: false, true, 'custom'
interactiveScrollbars boolean true Enables or disables scrolling while dragging Scrollbars.
eventPassthrough boolean true If true, iScrolls react to horizontal swipes only. Useful if Vertical Scrolling is used somewhere in the page.
