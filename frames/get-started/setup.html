Getting started
Introduction

Welcome to MetroUI!

MetroUI is a framework that lets you build your own apps or websites with an exciting Windows 8-like design. There are many Metro frameworks out there in the World Wide Web. Some of them are better, some of them are worse. But MetroUI's aim is to stick as close to the original as possible.

You will now learn everything that's possible with MetroUI, starting with setting up MetroUI for the first use and how to configure it to your needs. Then you will learn how to add components to your app or website.

So let's get started!

Basic Layout

Creating a page in MetroUI is really easy. First create a file called index.html.

Using the template on the right will give you a simple content-based page, with a title and space for MetroUI components.

It also adds required libraries and Stylesheets. You can change the paths if you need to.

 
<!DOCTYPE html>
<html>
  <head>
    <title>MetroUI Demo Page</title>
    <link rel="stylesheet" type="text/css" href="css/metroui.css" />
    <link rel="stylesheet" type="text/css" href="css/metroui.themes.css" />

    <script src="js/lib/wrapper.js"></script>
    <script src="js/lib/jquery.js"></script>
    <script src="js/lib/fastclick.js"></script>
    <script src="js/lib/iscroll.js"></script>
   <script src="js/MetroUI.js"></script>
  </head>
  <body>
    <!-- Optional Bars here -->
    <div class="pages">
      <div class="page" data-page="main">
      
        <div class="page-header">
          <!-- Pivot Title for Windows Phone Mode only -->
          <div class="pivot-title">Page Header</div>
          <div class="pivot-header">Page Title</div>
        </div>
        
        <div class="page-content">
          <div class="page-content-wrapper">
            <div class="scroll-content">
              <!-- Add Components here -->
            </div>
          </div>
        </div>
        
      </div>
    </div>
  </body>
</html>
Start the engine

When you're done with creating your page layout, you need to initalize MetroUI, or "start its engine". When doing this, we need several libraries, which are included with MetroUI. These must be added in the <head></head> before the MetroUI script.

You can either initialize MetroUI right within your page or you use an external JavaScript file, which you only need to include inside your page.

Which way you go is up to you. Just ensure you are doing it right before the closing </body> tag.

 
<!-- In-Page initialization -->
<script type="text/javascript">
  var app = new MetroUI();
</script>
