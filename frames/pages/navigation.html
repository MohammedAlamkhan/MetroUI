Description

Navigation is that thing that happens between Page A and Page B. Without Navigation you wouldn't be able to transition through pages.

Navigation is done through JavaScript. It intercepts links to pages (<a href="">...</a>) and loads their content in the current DOM, without clustering the history. It also enables smooth transition between pages and allows you to apply minimal code layout.

Links that have an external class are loaded in the default way, bringing you directly to the new page (e.g. if you want to navigate to Google or any other website that is not yours). A JavaScript method also allows you to load a page manually, either being in the current DOM, using the page identification string, or from an external file.

Attributes & Classes
Attribute Description
page Identification string of a page that exists in the current DOM
url URL of a file with minimal layout to load in the current DOM
Code
var app = new MetroUI();
app.loadPage(page);
// Loads page from current DOM

app.loadPageExternal(url);
// Loads external page from specified URL
